# Allied Voices API

This API powers the [front-end](https://github.com/nyette/allied-voices-client).

## Installation

```bash
$ npm install
```

## Modes

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
